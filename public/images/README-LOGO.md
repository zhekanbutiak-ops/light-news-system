# Логотип для Light News

Поклади сюди файл **logo.png** — він з’явиться:
- у шапці сайту (біля тексту «Light News») на десктопі;
- у прев’ю при поширенні посилання (Telegram, соцмережі, Google).

**Рекомендовані розміри:**
- Для шапки: висота 32–64 px, пропорції на твой вибір (наприклад 120×40 px або квадрат 64×64).
- Для соцмереж (og:image): **1200×630 px** — тоді прев’ю буде без обрізання.

Якщо logo.png відсутній, у шапці показується лише текст «Light News»; після додавання файлу картинка підхопиться автоматично.
