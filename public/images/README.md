# Fallback-зображення для новин

Якщо з RSS не вдається витягнути картинку (наприклад, для АрміяInform), підставляються ці файли:

- **fallback-news.jpg** — загальна картинка для новин (рекомендовано 800×450 або пропорційно).
- **fallback-front.jpg** — картинка для тематики «фронт» / військові новини (armyinform.com.ua тощо).

Покладіть сюди файли з такими іменами — вони будуть використовуватись автоматично. Якщо файлів немає, браузер покаже порожній блок або можна задати URL через змінні оточення `NEXT_PUBLIC_FALLBACK_NEWS_IMAGE` та `NEXT_PUBLIC_FALLBACK_FRONT_IMAGE`.
